#### 문제 상황

**신반포자이 예시**:
```
실제 분석: 91.3점 (S등급)  ← 정확
필터 검색: 80점 (A등급)   ← 틀림!
```

#### 해결 방법

**estimateGrade 함수 완전 재작성**:

```javascript
function estimateGrade(a) {
  // 1. 지역선호도 - LOCATION_PREFERENCE 사용
  let p1 = 70;
  for (let region in LOCATION_PREFERENCE) {
    if (dong && dong.includes(region)) {
      p1 = LOCATION_PREFERENCE[region];
      break;
    }
  }
  
  // 2. 핵심시설 - 역거리 기반
  let p2 = sd <= 300 ? 85 : sd <= 500 ? 75 : ...
  
  // 3. 교통 - scoreTransit과 동일
  let p3 = ...
  const busBonus = isSeoul ? 18 : isCap ? 12 : 3;
  p3 = Math.min(100, p3 * 0.75 + busBonus);
  
  // 4. 학군 - SCHOOL_SCORE 사용
  const p4 = SCHOOL_SCORE[gu] || 50;
  
  // 5. 편의시설 - 기본값
  const p5 = 55;
  
  // 6. 신축도 - 연도별 정확한 점수
  let p6 = yr >= 2023 ? 95 : yr >= 2020 ? 85 : ...
  
  // 7. 가격 수준 - PRICE_MAP + DB price
  let p7 = pr > 0 ? (1.4 - pr/guBase) / 0.8 * 100 : 50;
  
  // 가중 합산 (analyze()와 동일)
  const total = (p1*20 + p2*20 + p3*15 + p4*15 + p5*10 + p6*10 + p7*10) / 100;
  
  return total >= 80 ? 'S' : total >= 70 ? 'A' : ...
}
```
