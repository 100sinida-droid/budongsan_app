1. GitHub 연결 설정

이 저장소에 파일을 올릴 수 있도록 GitHub 정보를 입력하세요. 토큰은 이 브라우저에만 저장되며 어디에도 전송·공개되지 않습니다.

GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token → 대상 저장소 선택 → Repository permissions의 Contents 를 Read and write 로 설정 후 생성하세요.

2. 회차 업로드

작품을 고르고, 원고 파일(.md / .txt)을 여러 개 한꺼번에 올리세요. 파일명의 숫자로 회차 번호가 자동 인식됩니다. (예: 012.md → 12화)
